<h2>Programa a Desarrollar:</h2>
<h3>Graficador de espacio en memoria</h3>
<h4>Descripcion:</h4>
<p>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Este programa es un graficador de espacio donde se elije una carpeta de la cual se revisan 
diferentes datos para seguidamente mostrarlos en pantalla al usuario.
Estos datos serian:
<ul>
  &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;<li>El peso de la carpeta principal(la cual debe ser un directorio de carpetas) y seguidamente el peso de todas
  las carpetas anidadas a este mostrado graficamente con rectangulos hasta una profundidad de 6 anidamientos.</li>
  &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;<li>Las 10 carpetas con mayor cantidad de <strong>archivos</strong> de todo el directorio</li>
  &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;<li>Los 10 <strong>archivos</strong> mas pesados de todo el directorio</li>
</ul>
</p>
